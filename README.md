# 🍳 What to Eat

**今天吃什么？—— 打开冰箱就知道了。**

What to Eat 是一个个人专属的美食助手，基于冰箱现有食材智能推荐菜谱，并提供简易烹饪教学。

## ✨ 功能特色

**❄️ 三门智能冰箱**
- 保鲜层 / 冷冻层 / 储藏层，每层独立开门
- 食材健康度爱心系统（❤️ 正常 → 💛 将过期 → 🖤 紧急）
- 到期提醒，一目了然

**📖 智能菜谱推荐**
- 基于现有食材推荐可做菜品
- 可按食材过滤筛选
- 10+ 道家常菜谱，持续扩充

**👨‍🍳 烹饪教学**
- 流程图风格步骤教学
- 已完成的食材可打勾标记
- 所有食材显示 emoji 图标

**📝 饮食日记**
- 每日打卡记录
- 食材用料详情（点击展开）
- 每日热量统计

**👤 个人主页**
- 常做菜自动统计
- 收藏 / 想学 / 讨厌食材管理
- 设置面板：头像上传、昵称修改、数据备份

## 🎨 设计风格

暖色调像素风，灵感来自《星露谷物语》。主色：南瓜橙 #E8762D、暖米白 #FFF8ED、木边框 #8B6914。

## 🚀 快速使用

直接访问 GitHub Pages 链接，用 iPhone Safari 打开：

1. 点击底部 **分享** 按钮
2. 选择 **添加到主屏幕**
3. 像原生 App 一样使用

## 🛠 技术栈

- 纯前端 SPA（HTML + CSS + JavaScript）
- 本地数据存储（localStorage）
- PWA 支持：可离线使用，添加到主屏幕
- 暖木色像素风 UI

## 📂 文件结构

```
WhatToEat.html   — 主应用
manifest.json    — PWA 配置文件
sw.js            — Service Worker 离线缓存
icon-1024.png    — App 图标
icon-design.html — 图标设计工具
```
